# ServixOS Web

A production-ready **Next.js + TypeScript** application for service-business operations, including:

- Client, job, invoice, payment, and team management flows
- AI insights and advisor experiences
- Global state via Context API
- TanStack Query provider setup
- SEO metadata, robots, and sitemap support
- Offline/slow-network UX handling

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Production build
- `npm run start` - Start production server
- `npm run lint` - Lint the project
